Ferramentas Utilizadas no Projeto

Documentação técnica das tecnologias e recursos do site.

Linguagens de Programação

HTML5

  • Descrição: Linguagem de marcação utilizada para estruturar o conteúdo.

  • Uso: Construção de elementos semânticos (header, nav, main, footer).

  • Recursos: Meta tags para dispositivos móveis e SEO.

CSS3

  • Descrição: Linguagem de estilo para a apresentação visual.

  • Uso: Layouts responsivos e efeitos visuais.

  • Recursos: Flexbox, Grid, Media Queries e Transições.

Ferramentas de Desenvolvimento

Visual Studio Code

Ambiente principal de desenvolvimento utilizando extensões como Live Server e Prettier.

Git & GitHub

Controle de versão e hospedagem do código fonte via GitHub Pages.

Hospedagem e Deploy

O site está hospedado no GitHub Pages, garantindo SSL gratuito e deploy automático.

Recursos Adicionais

  • Imagens de tipo jpg: Capas dos álbuns otimizadas para web.

  • Audios de tipo mp3: Integração de reprodutores do YouTube e demais redes sociais.

  • Arquivos de tipo txt: Arquivos de texto para documentação e configuração.

  • Arquivo .gitignore: Arquivo de configuração de arquivos especificos que devem ser ignorados pelo Git.

Considerações Finais

  • Custo: Soluções 100% gratuitas.

  • Performance: Otimizado para carregamento rápido.

  • Compatibilidade: Suporte a navegadores modernos.
